Классы точек стационарности

Поскольку нейросети в общем случае представляют собой сложные нелинейные функции, то их оптимизация их весов невозможна аналитически и всегда производится численными градиентными методами. Далее предполагается знакомство читателя с градиентными методами оптимизации из первой части учебника, и будут приведены улучшения этих методов для настройки нейросетей.

Мы минимизируем функцию потерь L(w)L(w)L(w) по весам модели w∈RKw\in\mathbb{R}^Kw∈RK:

L(w)→min⁡wL(w)\to\min_wL(w)→wmin​

Разложение Тейлора 2-го порядка для L(w)L(w)L(w) имеет вид:

L(w^)≈L(w)+ΔwT∇L(w)+12ΔwT∇2L(w)Δw,L(\hat{w})\approx L(w) + \Delta w^T \nabla L(w)+\frac{1}{2}\Delta w^T \nabla^2 L(w) \Delta w,L(w^)≈L(w)+ΔwT∇L(w)+21​ΔwT∇2L(w)Δw,

где

  • Δw=w^−w\Delta w = \hat{w}-wΔw=w^−w,

  • ∇L(w)∈RK\nabla L(w)\in\mathbb{R}^K∇L(w)∈RK - градиент функции (вектор из частных производных по каждому весу);

  • ∇2L(w)∈RK×K\nabla^2 L(w)\in\mathbb{R}^{K\times K}∇2L(w)∈RK×K - матрица Гессе (Гессиан) из вторых производных {L(w^)∂wi∂wj}i,j\left\{\frac{L(\hat{w})}{\partial w_i\partial w_j}\right\}_{i,j}{∂wi​∂wj​L(w^)​}i,j​.

Если раскладывать функцию потерь в стационарной точке w∗w^*w∗ (в которой градиент равен нулю, т.е. ∇L(w)=0\nabla L(w)=0∇L(w)=0), то разложение Тейлора примет вид:

L(w^)≈L(w∗)+12ΔwT∇2L(w∗)Δw,(2)L(\hat{w})\approx L(w^*)+\frac{1}{2}\Delta w^T \nabla^2 L(w^*) \Delta w, \tag{2}L(w^)≈L(w∗)+21​ΔwT∇2L(w∗)Δw,(2)

Гессиан, будучи симметричной матрицей, имеет ортонормированный базис из собственных векторов u1,...uKu_1,...u_Ku1​,...uK​ с собственными значениями λ1,...λK\lambda_1,...\lambda_Kλ1​,...λK​:

∇2L(w∗)ui=λiui,i=1,2,....K.\nabla^2 L(w^*) u_i = \lambda_i u_i,\quad i=1,2,....K.∇2L(w∗)ui​=λi​ui​,i=1,2,....K. uiTuj={1, если i=j,0, если i≠j.u_i^T u_j = \begin{cases} 1, \text{ если } i=j, \\ 0, \text{ если } i\ne j. \end{cases}uiT​uj​={1, если i=j,0, если i=j.​

Разложим Δw=w^−w∗\Delta w = \hat{w}-w^*Δw=w^−w∗ по базису из собственных векторов:

Δw=∑kαkuk\Delta w = \sum_k \alpha_k u_kΔw=k∑​αk​uk​

Подставив это разложение в (2), получим:

L(w^)−L(w∗)≈12(∑kαkuk)T∇2L(w∗)(∑kαkuk)=12(∑kαkuk)T(∑kαkλkuk)=12∑kλkαk2.\begin{align*} L(\hat{w})-L(w^*)&\approx\frac{1}{2}\left(\sum_k \alpha_k u_k\right)^T \nabla^2 L(w^*) \left(\sum_k \alpha_k u_k\right) \\ &=\frac{1}{2}\left(\sum_k \alpha_k u_k\right)^T \left(\sum_k \alpha_k \lambda_k u_k\right) \\ &=\frac{1}{2}\sum_k \lambda_k \alpha_k^2. \end{align*}L(w^)−L(w∗)​≈21​(k∑​αk​uk​)T∇2L(w∗)(k∑​αk​uk​)=21​(k∑​αk​uk​)T(k∑​αk​λk​uk​)=21​k∑​λk​αk2​.​

Отсюда видно, что точка стационарности w∗w^*w∗ может быть:

  • точкой минимума, когда все λi\lambda_iλi​ положительны (Гессиан - положительно определённая матрица);

  • точкой максимума, когда все λi\lambda_iλi​ отрицательны (Гессиан - отрицательно определённая матрица);

  • точкой перегиба, когда часть собственных значений положительна, а часть отрицательна.

Точки перегиба часто возникают при минимизации невыпуклых функций и затрудняют работу градиентных методов оптимизации, поскольку градиент в их окрестности становится малым и обучение происходит медленно. Чтобы с этим бороться, можно использовать стохастический градиентный спуск с инерцией.
