Аналитическое решение для линейной регрессии

Веса линейной регрессии находятся из принципа минимизации наименьших квадратов (МНК, ordinary least squares, OLS):

L(w)=∑n=1N(xnTw−yn)2→min⁡wL(\mathbf{w})=\sum_{n=1}^N (\mathbf{x}_n^T\mathbf{w}-y_n)^2\to\min_{\mathbf{w}} L(w)=n=1∑N​(xnT​w−yn​)2→wmin​

Этот критерий по w\mathbf{w}w является выпуклым как суперпозиция линейной и выпуклой функции.

Тип векторов

Все вектора в этом разделе, как и глобально в книге, будем считать векторами-столбцами (а не строками). Это важно для понимания аналитических выкладок.

Будет ли суперпозиция двух выпуклых функций выпуклой?

Суперпозиция двух выпуклых функций может быть выпуклой, например, f(g(x))=(∣x∣+1)2f(g(x))=(\vert x \vert+1)^2f(g(x))=(∣x∣+1)2 для

g(x)=∣x∣+1,f(x)=x2g(x)=\vert x \vert + 1,\quad f(x)=x^2g(x)=∣x∣+1,f(x)=x2

Но может быть и невыпуклой, как f(g(x))=(∣x∣−1)2f(g(x))=(\vert x \vert-1)^2f(g(x))=(∣x∣−1)2 для

g(x)=∣x∣−1,f(x)=x2g(x)=\vert x \vert - 1,\quad f(x)=x^2g(x)=∣x∣−1,f(x)=x2

Однако суперпозиция линейной и выпуклой функции всегда выпукла - докажите самостоятельно.

Поэтому не только необходимым, но и достаточным условием минимума потерь будет покомпонентное равенство нулю градиента функции потерь. Причем, также в силу выпуклости функции, будет гарантия, что найденный минимум будет глобальным минимумом функционала, т.е. обеспечивать наименьшее значение функции потерь среди всех возможных.

Мы говорим про градиент, а не про производную, поскольку при дифференцировании скалярной функции L(w)∈RL(\mathbf{w})\in\mathbb{R}L(w)∈R по вектору w∈RD+1\mathbf{w}\in \mathbb{R}^{D+1}w∈RD+1 получим вектор частных производных по каждой компоненте вектора w\mathbf{w}w:

∇L(w)=(∂L∂w0∂L∂w1∂L∂w2⋯∂L∂wD)\nabla L(\mathbf{w})= \begin{pmatrix} \frac{\partial L}{\partial w_0} \\ \frac{\partial L}{\partial w_1} \\ \frac{\partial L}{\partial w_2} \\ \cdots \\ \frac{\partial L}{\partial w_D} \\ \end{pmatrix}∇L(w)=​∂w0​∂L​∂w1​∂L​∂w2​∂L​⋯∂wD​∂L​​​

Найдем решение аналитически:

∇L(w^)=2∑n=1Nxn(xnTw^−yn)=0\nabla L(\widehat{\mathbf{w}})=2\sum_{n=1}^{N}\mathbf{x}_{n}\left(\mathbf{x}_{n}^{T}\widehat{\mathbf{w}}-y_{n}\right)=0∇L(w)=2n=1∑N​xn​(xnT​w−yn​)=0 (∑n=1NxnxnT)w^=∑n=1Nxnyn\left(\sum_{n=1}^{N}\mathbf{x}_{n}\mathbf{x}_{n}^{T}\right)\widehat{\mathbf{w}}=\sum_{n=1}^{N}\mathbf{x}_{n}y_{n}(n=1∑N​xn​xnT​)w=n=1∑N​xn​yn​

Перепишем условие, используя обозначения для матрицы объекты-признаки X∈RN×DX\in\mathbb{R}^{N\times D}X∈RN×D и вектора откликов Y∈RNY\in \mathbb{R}^NY∈RN:

XTXw^=XTYX^{T}X\widehat{\mathbf{w}}=X^T YXTXw=XTY

Откуда получаем аналитическое итоговое решение для линейной регрессии:

w^=(XTX)−1XTY\widehat{\mathbf{w}}=(X^{T}X)^{-1}X^{T}Yw=(XTX)−1XTY

Обратим внимание, что решение не будет существовать, если матрица XTXX^T XXTX вырождена, что эквивалентно тому, что ранг матрицы rg(XTX)=rg(X)<D\text{rg}(X^T X)=\text{rg}(X)<Drg(XTX)=rg(X)<D.

Задача

Докажите, что rg(XTX)=rg(X)\text{rg}(X^T X)=\text{rg}(X)rg(XTX)=rg(X) для любой матрицы XXX.

Последнее, в свою очередь, означает линейную зависимость между признаками, т.е. что найдётся такой набор весов α=[α0,α1,...αD]\bm{\alpha}=[\alpha_0,\alpha_1,...\alpha_D]α=[α0​,α1​,...αD​], что

xTα=0∀x (выполнено для любых векторов признаков x)\mathbf{x}^T \bm{\alpha}=0 \quad \forall \mathbf{x} \,(\text{выполнено для любых векторов признаков } \mathbf{x})xTα=0∀x(выполнено для любых векторов признаков x)

В этом случае один из признаков лишний, поскольку может быть получен как линейная комбинация других признаков, а само решение неоднозначно, поскольку если w^\hat{\mathbf{w}}w^ - решение, то и w+kα\mathbf{w}+k\bm{\alpha}w+kα - тоже решение для любого k∈Rk\in \mathbb{R}k∈R, поскольку

y^(x)=xTw^=xTw^+0=xTw^+kxTα=xT(w+kα)\hat{y}(\mathbf{x})=\mathbf{x}^T\hat{\mathbf{w}}=\mathbf{x}^T\hat{\mathbf{w}}+0=\mathbf{x}^T\hat{\mathbf{w}}+k\mathbf{x}^T\bm{\alpha}=\mathbf{x}^T(\mathbf{w}+k\bm{\alpha})y^​(x)=xTw^=xTw^+0=xTw^+kxTα=xT(w+kα)

Следовательно, если матрица XTXX^T XXTX необратима и аналитическая оценка весов не определена, то нужно уменьшить число признаков либо через отбор признаков, либо через снижение размерности.
