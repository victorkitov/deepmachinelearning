Блендинг моделей

Для повышения точности прогнозов часто используют ансамбли моделей (model ensemble), называемые также композициями моделей. Для этого используют прогнозы MMM базовых моделей f1(x),...fM(x)f_{1}\left(x\right),...f_{M}\left(x\right)f1​(x),...fM​(x), а итоговый прогноз строят через агрегирующей модель (называемую также мета-моделью) G(f1(x),...fM(x))G(f_{1}\left(x\right),...f_{M}\left(x\right))G(f1​(x),...fM​(x)). Таким образом, прогнозы базовых моделей выступают признаками для агрегирующей модели .

Если агрегирующая модель линейна, то такой подход называется блендингом моделей (model blending):

y^(x)=f(x)Tβ^\widehat{y}(x)=f\left(x\right)^{T}\widehat{\beta}y​(x)=f(x)Tβ​

Веса β\betaβ этой модели настраиваются через линейную регрессию на отдельной обучающей выборке - не на той, на которой настраивались параметры базовых моделей, чтобы избежать переобучения.

Веса можно настраивать обычным способом, а можно ввести дополнительные требования из логики задачи:

{∑n=1N(f(xn)Tβ−yn)2+λ∑m=1M(βm−1M)2→min⁡ββm≥0,m=1,2,...M.(1)\begin{cases} \sum_{n=1}^{N}\left(f\left(x_{n}\right)^{T}\beta-y_{n}\right)^{2}+\lambda\sum_{m=1}^{M}\left(\beta_{m}-\frac{1}{M}\right)^{2}\to\min_{\beta}\\ \beta_{m}\ge0,\quad m=1,2,...M. \end{cases} \tag{1}⎩⎨⎧​∑n=1N​(f(xn​)Tβ−yn​)2+λ∑m=1M​(βm​−M1​)2→minβ​βm​≥0,m=1,2,...M.​(1)

Вместо L2-регуляризации рекомендуется использовать приближение весов к равномерным (1M,1M,...1M)(\frac{1}{M},\frac{1}{M},...\frac{1}{M})(M1​,M1​,...M1​). Тогда для больших λ\lambdaλ модель будет сдвигаться к равномерному усреднению прогнозов базовых моделей, а не константному прогнозу.

Дополнительно требуется неотрицательность весов, поскольку базовые модели, пытаясь предсказывать отклик, должны получиться положительно с ним скоррелированными.

Учет смещения

Если есть подозрение, что базовые модели систематически переоценивают или недооценивают прогноз, то в агрегирующую модель можно добавить смещение β0\beta_0β0​.
